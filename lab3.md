# Лабораторная работа №3 – Бизнес-процессы предметной области (AS-IS)
## Задание №1. Определение процессов системы
### Основной процесс: Управление библиотечным фондом

**Цель процесса:** Обеспечение эффективного учета и использования библиотечного фонда

**Инициатор:** Руководство библиотеки

**Входные данные:**

  * Каталог литературы

  * Данные о поставщиках

  * Запросы читателей

  * Бюджетные средства

**Выходные данные:**

  * Обновленный каталог

  * Отчеты о работе

  * Статистика использования

**Участники:**

  * Библиотекари

  * Читатели

  * Поставщики

  * Руководство

**Ожидаемый результат:**

  * Актуальный библиотечный фонд

  * Удовлетворенные запросы читателей

  * Эффективное использование ресурсов

### Вспомогательный процесс 1: Учет читателей

**Цель процесса:** Ведение актуальной базы данных пользователей

**Инициатор:** Библиотекари

**Входные данные:**

  * Личные данные читателей

  * Запросы на регистрацию

  * Информация об изменениях

**Выходные данные:**

  * Обновленная база читателей

  * Статистика посещаемости

**Участники:**

  * Читатели

  * Библиотекари

  * Администраторы системы

**Ожидаемый результат:**

  * Актуальная база пользователей

  * Корректные учетные записи

### Вспомогательный процесс 2: Контроль выдачи литературы

**Цель процесса:** Обеспечение своевременного возврата и корректного учета выданной литературы

**Инициатор:** Библиотекари

**Входные данные:**

  * Запросы на выдачу

  * Информация о наличии

  * Данные о читателях

**Выходные данные:**

  * Журналы выдачи

  * Отчеты о задолженностях

  * Статистика использования

**Участники:**

  * Читатели

  * Библиотекари

  * Система учета

**Ожидаемый результат:**

  * Корректный учет выданной литературы

  * Своевременный возврат материалов
## Задание №3. Анализ проблем процесса
|Возможная проблема|Причина|Последствие|
|---|---|---|
|Дублирование ввода данных|Ручной учет в разных журналах|Ошибки в данных, расхождения|
|Задержки при поиске литературы|Отсутствие единой системы каталогизации|Потеря времени, недовольство читателей|
|Ручные операции при учете|Отсутствие автоматизации|Ошибки в учете, потеря данных|
|Сложность контроля сроков|Отсутствие автоматизированного напоминания|Накопление задолженностей|
|Избыточность бумажной документации|Отсутствие электронного документооборота|Затраты на хранение, риск потери данных|
|Сложность формирования отчетов|Ручной сбор данных|Неактуальная информация, ошибки|
|Отсутствие единого доступа|Разрозненные системы учета|Неэффективное управление фондом|
